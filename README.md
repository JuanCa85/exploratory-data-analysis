# Análisis exploratorio de datos

## Descripción del proyecto

Proyecto enfocado en la limpieza, transformación y análisis exploratorio de un conjunto de datos para detectar patrones, valores atípicos y relaciones entre variables.

### Objetivo

Obtener insights relevantes a partir del análisis exploratorio de datos.

**Herramientas utilizadas**

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

**Proceso del análisis**

1. Limpieza de datos.
2. Tratamiento de valores nulos.
3. Análisis de distribuciones.
4. Detección de outliers.
5. Visualización de relaciones entre variables.

**Resultados principales**

- Identificación de patrones clave.
- Detección de anomalías en los datos.
- Conclusiones basadas en visualizaciones.

**Habilidades demostradas**

- Limpieza de datos
- Análisis exploratorio
- Visualización
- Interpretación de resultados
